# 🎯 주요 기능 상세

## 1. 레이아웃 그리기

| 기능 | 설명 |
|------|------|
| **사각형** | 드래그로 크기 조절, 회전 가능 |
| **다각형** | 복잡한 형상 지원, Shoelace 공식으로 면적 계산 |

---

## 2. 스냅 기능

| 기능 | 설명 |
|------|------|
| **그리드 스냅** | 10px 단위 그리드에 정렬 |
| **객체 스냅** | 다른 객체의 모서리/중심에 정렬 |

---

## 3. 단위 변환

| 변환 | 비율 |
|------|------|
| **px → mm** | 1px = 0.264583mm |
| **mm → m** | 1mm = 0.001m |

> 면적도 자동 변환됩니다 (px² → mm² → m²)

---

## 4. 줌/팬

| 기능 | 조작 방법 |
|------|----------|
| **줌 인/아웃** | 마우스 휠, 툴바 버튼 (0.1x ~ 10x) |
| **팬** | 스페이스바 + 드래그 |

---

## 5. 동적 텍스트 크기 조정

| 기능 | 설명 |
|------|------|
| **줌 보정** | 줌 아웃 시 텍스트가 너무 작아지지 않도록 자동 확대 (최대 3배) |
| **컨테이너 기반** | 레이아웃/머신 크기에 맞춰 텍스트가 넘치지 않도록 조정 |
| **범위 제한** | 최소 8px ~ 최대 48px 범위 내 자동 조정 |

> 줌 레벨 변경 시 텍스트가 항상 가독성을 유지하며, 객체 크기에 맞게 최대한 크게 표시됩니다.

---

## 6. 그룹화 및 선택

| 기능 | 설명 |
|------|------|
| **그룹화** | 여러 객체(사각형, 다각형, 머신) 선택 후 그룹화 |
| **그룹 이동** | 그룹 전체를 하나의 단위로 실시간 이동 |
| **중첩 선택** | 첫 클릭으로 그룹 전체 선택, 두 번째 클릭으로 개별 객체 선택 |
| **드래그 선택** | 빈 캔버스 영역에서 드래그하여 여러 객체 동시 선택 |
| **ESC 키** | 선택 전체 취소 |
| **계층적 렌더링** | Group 컴포넌트를 사용, 텍스트 라벨도 드래그 시 함께 이동 |

---

## 7. 속성 편집 및 잠금 (New)

| 기능 | 설명 |
|------|------|
| **직접 편집** | Properties Panel에서 X, Y, Width, Height 값을 직접 입력하여 수정 |
| **속성 잠금** | 각 속성 옆의 🔒 아이콘으로 값 고정 (편집/드래그 방지) |
| **Area Lock** | 사각형의 면적(Area)을 잠그면, 가로/세로 변경 시 반대 차원이 자동 조절되어 면적 유지 |
| **초기 Area** | 레이아웃 생성 시 즉시 면적 계산 및 표시 |
